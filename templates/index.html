Задание

  1. Изучить инфраструктуру

    2. Инфраструктура представлена на рисунке 1.

    Рисунок 1 - Инфраструктура

    Серверная часть уже реализована и состоит из двух компонентов:

    • Server, реализующий backend системы;
    • Database, база данных, накапливающая информацию в системе.

    Взаимодействие сервера и клиента осуществляется через API в сети Интернет.

    Client – клиентское приложение, которое может быть реализовано как веб-приложение (работает из браузера), так и как Python-приложение.

    Клиент и сервер могут связываться только по API. Прямое взаимодействие исключается политикой безопасности.

  1. Разработать клиентское приложение

    2. Вашей следующей задачей является разработка клиентского приложения (frontend). Вы можете реализовать один из вариантов на ваш выбор:

    1. Веб-клиент. В этом случае клиентское приложение реализуется набором HTML, JS и CSS-файлов, которые позволяют сформировать и отправить запрос к API, а также получить и отобразить ответ.
    2. Python-приложение. Представляет собой py-файл (а также любые дополнительные файла), который позволяет сформировать и отправить запрос к API, а также получить и отобразить ответ.
  1. Отправить ответ

    2. Теперь, когда ваше приложение готово, сформируйте и отправьте запрос к API. В запросе обязательно должны быть указаны ваши имя и фамилия.

    Для получения дополнительных баллов нужно указать следующую информацию:

    • Баллы по математике ученика с ID = 7;
    • Наибольшая сумма баллов по русскому языку и математике;
    • Средний балл по русскому языку (только целая часть числа).

    Каждый ответ дает дополнительный балл.
